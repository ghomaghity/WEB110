Variable Typing

Typing refers to what type of data can be held inside of a variable. Variables in JavaScript are dynamically typed, meaning they can store whatever type can be assigned to them without issue. Java is statically typed. This means that whatever type a varaible is designated to hold when declared is the only thing it will ever be able to hold.

Compilation Methods

Java is just in time (JIT) compiled into bytecode. This bytecode is then interpreted using the JVM. JavaScript can be both JIT compiled and interpreted in real time.


Example

Here's an example hello world program in Java:
public static void main(String[] args) {
    System.out.println("Hello world!");
}
And here's an example hello world program in JavaScript:
console.log("Hello world!");
